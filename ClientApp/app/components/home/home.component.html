Kendo UI for Angular Grid consuming WebAPI
with DataSourceRequest and ToDataSourceResult()

Welcome to your new single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular 2 and TypeScript for client-side code
  • Webpack for building and bundling client-side resources
  • Kendo UI for Angular for layout and styling

Kendo UI - related configuration

  • Web API endpoint, parsing the requests and returning the response using the Kendo UI for ASP.NET MVC DataSourceRequest and ToDataSourceResult(). For more information about including Kendo UI in an ASP.NET Core project, refer to this article.
  • Kendo UI for Angular Grid documentation is available here.
  • Kendo UI for Angular DataQuery toDataSourceRequestString() method converts the request into string, comparable with UI for ASP.NET MVC DataSourceRequest format.
